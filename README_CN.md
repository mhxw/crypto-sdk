# Crypto-SDK

[English](README.md) | 中文

Crypto-SDK是针对主流区块链的转账和查询的js工具。

------

## 安装

- 克隆此项目

```shell script
git clone https://github.com/mhxw/crypto-sdk.git
```

- 安装依赖

```bash
yarn install
```

## 案例

- 运行测试案例

```bash
node eth.js
```

- 启动nodejs

```bash
node app.js
```

## API

- 查询余额

```bash
curl http://localhost:8082/getBalance?account=0x01f933904539fe8662c48392ee31c0afcf98758e
```
